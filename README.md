# Speeding Fines Tracker

This Java program records monthly speeding fines using a two-dimensional array and displays the sum, highest, and lowest speeding fines in the array.

## Features

- **Data Recording**: Records monthly speeding fines for different cities.
- **Report Generation**: Displays a report showing the speeding fines for each city and month.
- **Statistics Calculation**: Calculates and displays the sum, highest, and lowest speeding fines in the array.

## Skills and Lessons Learned

- **Java Programming**: Practiced array manipulation and control structures.
- **Data Structures**: Gained experience using two-dimensional arrays to store and process data.
- **Algorithms**: Implemented algorithms to find the sum, maximum, and minimum values in an array.
- **Console Output**: Learned to format and display data in a readable manner using console output.

## Program Notes

- **Speeding Fines Array**: The program uses a two-dimensional array to store speeding fines for different cities (Johannesburg, Durban, Cape Town, Port Elizabeth) across three months (January, February, March).
- **Report Formatting**: The program prints a formatted report with city names, month names, and the corresponding speeding fines.
- **Statistics Calculation**: The program calculates the sum, highest, and lowest speeding fines from the array and displays the results.
